### Short Description (max 132 chars)
```

```

### Detailed Description

```
Take full control of YouTube's keyboard shortcuts.

YouTube Keystrokes Blocker lets you individually enable or disable each of YouTube's hotkeys. Accidentally triggering shortcuts while scrolling or chatting? Block only the ones that cause problems — keep the rest working exactly as you expect.

── KEY FEATURES ──

✅ 30+ independent hotkey toggles — control each key individually
✅ Settings open instantly with Alt+Shift+H or via the toolbar icon
✅ Smart input detection — won't block keys while typing in search, comments, or chat
✅ Settings persist across YouTube's SPA navigation
✅ Zero data collection — all settings stored locally in your browser
✅ Fully open-source on GitHub

── HOTKEYS YOU CAN CONTROL ──

• Numbers (0–9) — Jump to video position
• Spacebar — Play/Pause
• Arrow Keys — Timeline and volume navigation
• Ctrl + Left/Right — Chapter navigation
• F — Fullscreen toggle
• M — Mute/Unmute
• K — Play/Pause (alternative)
• J / L — Seek ±10 seconds
• T — Theatre mode toggle
• C — Captions toggle
• I — Mini player toggle
• Shift + < / > — Playback speed
• Period / Comma — Frame-by-frame navigation
• Home / End — Jump to start/end

── KEYBOARD SHORTCUT ──

Press Alt+Shift+H (Windows/Linux) or Cmd+Shift+H (Mac) on any YouTube page to instantly open the settings popup.

── PRIVACY ──

This extension only runs on youtube.com. It collects absolutely no data. All settings are stored locally using chrome.storage.sync and never leave your device.

── PERMISSIONS ──

• storage — save your hotkey preferences locally
• youtube.com host permission — run only on YouTube pages

── OPEN SOURCE ──

Full source code: https://github.com/Life-Experimentalist/Youtube-Keystrokes-Blocker

Also available as a userscript for Tampermonkey/Violentmonkey:
https://greasyfork.org/en/scripts/563265-disable-youtube-hotkeys-with-modern-settings-page
```
